Overwatch
Visual studio, S7.net wrapper, c# .net, GIMP (for design of icons etc.)

Need for new application rose when plant workers didn’t have actual information about what material they are going to process next based on the conveyor system that was coming from the previous plant (waste processing). Without the possibility to use OPC server a s7.net wrapper was used to obtain information directly from registers of Siemens PLC and it was used to display information on screen for the workers to know what kind of material is being processed. It helped the company to be more efficient.

Process Control Tool
Visual studio, C#.net, SQL, Microsoft SQL Management studio, KepServer (Dataloger), PdfSharp (to fill PDF form with parameters for printing), hashing of password to secure password in hash form in database

Application that used OPC server (KepServer Datalogger) to store all parameters of whole lines during “save of parameters” operation to SQL database. The whole application uses login with password verification to permit access only to authorized personal to accept or reject new changes to the parameters done on the press line. User with operator status is capable only to add comment to the changes made by him with parameters and view (or print in PDF) master copy of approved parameters. Engineering can also access history of changes to parameters (for audit purpose) and is also capable to track process parameters during each production (like press force for example).
